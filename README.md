# Wayground Quiz Answer Viewer

## Description

A simple tool to view quiz answers on Wayground.

## Features

- View quiz answers in a simple and clear interface
- Lightweight and easy to use
- Fast dependency management with `uv`

## Tech Stack

- Python
- Streamlit
- uv

## Requirements

- Python 3.9+
- uv

## Project Structure

```bash
├── main.py
├── crawler.py
├── pyproject.toml
├── uv.lock
└── README.md
```

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/hvd1026/wayground-answer
   cd wayground-answer

   ```

2. Install dependencies:
   ```bash
    uv sync
   ```
3. Run the application:
   ```bash
   uv run streamlit run main.py
   ```
